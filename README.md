Meizhi-Crawler
==============

A web crawler for http://meizhi.im. Enjoy!!!

## Install

```
git clone https://github.com/saintwinkle/meizhi-crawler.git
cd meizhi-crawler
(sudo) npm install
```

## Run

```
node app.js
```

Images will be downloaded in `meizhi-crawler` folder.
