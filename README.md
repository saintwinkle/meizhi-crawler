Meizhi-Crawler
==============

A web crawler for http://meizhi.im.

## Install

```
git clone https://github.com/saintwinkle/meizhi-crawler.git
cd meizhi-crawler
(sudo) npm install
```

## Run

```
node app.js (app_normal.js)
```

Images will be saved in `images` folder. Enjoy!
